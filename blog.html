What are the different ways to select an element in the DOM?
  • getElementById() and select ID
  • getElementsByClassName(), class name select and Returns an HTMLCollection.
  • getElementsByTagName(), Selects tag name and Returns an HTMLCollection.
  • querySelector(), first matching element based on a CSS selector.
  • querySelectorAll() , Selects all matching elements based on a CSS selector
What is the difference between innerHTML, innerText, and textContent ?
  • innerHTML ,HTML content and including tags

What is event delegation in the DOM?
Event Delegation একটি মাত্র ইভেন্ট লিসেনার প্যারেন্ট এলিমেন্টে যোগ করা, যা তার চাইল্ড এলিমেন্টগুলোর ইভেন্ট হ্যান্ডেল করতে পারে
What is event bubbling in the DOM?
Event Bubbling হলো DOM ইভেন্ট প্রোপাগেশনের একটি ধাপ, যেখানে কোনো চাইল্ড এলিমেন্টে ইভেন্ট ঘটলে, সেটি তার প্যারেন্ট এবং আরও উপরের এলিমেন্টগুলোর কাছেও পৌঁছে যায়।
